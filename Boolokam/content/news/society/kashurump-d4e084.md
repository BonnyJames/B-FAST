''ദേ .., നിങ്ങള്‍ ഒന്നിങ്ങൊട്ട് ഓടി വന്നേ ...!''

ഭാര്യയുടെ ആ അലര്‍ച്ച .., എന്നെ ഞെട്ടിപ്പിച്ചു കളഞ്ഞാണ് എഴുന്നെല്പ്പിച്ചത് ...

''ഇതെന്തൊരു മാരണം ..., മനുഷ്യനെ ഒന്ന് കിടന്നുറങ്ങാനും സമ്മതിക്കില്ലേ ....?''
ഡിസംബറിലെ നല്ല മഞ്ഞു പുതച്ച തണുത്ത സുഖാലസ്യം .., വിട്ടുണരാന്‍ .., ശരീരവും .., മനസ്സും ഒരുപോലെ മടിക്കുന്ന സമയം ...!

''എഴുന്നേറ്റ് വായോ .., മനുഷ്യനേ .....''!, ഭാര്യ വിടാന്‍ ഭാവമില്ല ...!

'' പപ്പാ .., വേഗം വായോ ...''!, ചെക്കന്റെ കാറലാണെങ്കില്‍ അതിനും മേലേ ...!

ആലസ്യത്തോടെ പൂമുഖത്ത് എത്തിയപ്പോള്‍ .., ഭാര്യയും .., മക്കളും എന്തോ അത്ഭുതം കാണുന്ന മാതിരി .., താഴോട്ട് നോക്കി ക്കൊണ്ട് നില്‍ക്കുന്നു ..!

''എന്താ ..?''

''ദേ .., നിങ്ങള് .. ഒന്ന് നോക്ക്യേ മനുഷ്യനേ ...''?

''എന്തൂട്ട്...?''

''ഈ മനുഷ്യന് .., കണ്ണും കാണത്തില്ലേ .., ? താഴൊട്ട് നോക്കൂന്നേ ...''?

എനിക്കവിടെ പ്രത്യേകിച്ചൊന്നും കാണാന്‍ കഴിഞ്ഞില്ല .., കുറെ ഉറുമ്പുകള്‍ വരി വരിയായി പോകുന്നുണ്ട് ..!

''അത് കുറച്ച് ഉറുമ്പുകള്‍ അല്ലേ ...?, അതിനാണോ നീ കാറി .., കൂവി .., നാട്ടുകാരെ മൊത്തം ഉണര്‍ത്തിയത് ...?'', അത് ഇരതേടി പോകുന്നതായിരിക്കും ...''!

'' മനുഷ്യാ തോക്കില്‍ കയറി വെടി വെക്കുന്നതിനു മുന്‍പ് നിങ്ങളൊന്നു സൂക്ഷിച്ചു നോക്ക്യേ..., ഇത് കാശുറുമ്പുകളാ ......, ഇത് വന്നാല്‍ വീട്ടിലേക്ക് കാശ് വരൂന്നാ ...''!

കാശെന്ന് കേട്ടതേ .., എന്റെ ഉറക്കം അമ്പത് മൈല്‍ സ്പീഡില്‍ എങ്ങോട്ടോ ഓടിപ്പോയി

''എന്നാപ്പിന്നെ നീ കുറച്ചു പഞ്ചസാര എടുത്തോണ്ട് വാടി ...., നമുക്ക് കുറച്ചത് അകത്തോട്ട് വിതറി വിടാം .., !, വഴി തെറ്റി അതെങ്ങാനും അടുത്ത വീട്ടിലോട്ട് പോവണ്ടാ ..?!

എന്റെ ആ അതിബുദ്ധിയില്‍ ഭാര്യക്കും അതോടൊപ്പം എനിക്കും ആത്മാഭിമാനം തോന്നി

അവള്‍ എന്നൊയൊന്നു നോക്കി ...'' നിങ്ങള്‍ക്ക് ഇത്രയും ബുദ്ധിയുണ്ടോ .., എന്നായിരുന്നു ആ നോട്ടത്തിന്റെ അര്‍ത്ഥം ...!

പഞ്ചാര എന്ന് കേട്ടതേ .., മൂത്തോന്‍ അടുക്കളേലൊട്ട് പാഞ്ഞു .., കാശ് വരുന്നതോന്നുമല്ല അവനെ ഓടിച്ചത് ...,
പഞ്ചാര ഭരണിയാണ് .., രണ്ടു പിടി വായിലിടാന്‍ കിട്ടിയ സന്ദര്‍ഭമാണ് ...

പക്ഷേ .., ഇപ്പോഴത്തെ സിറ്റ്വേഷനില്‍ അതൊന്നും ചിന്തിക്കാനുള്ള സമയമല്ല.., കാശുറുമ്പല്ലേ .., കാശും കൊണ്ട് വന്നിരിക്കുന്നത് ...?

''ആരാപ്പാ .., നമുക്കിപ്പൊ കാശ് കൊണ്ട് വന്ന് തരാന്‍ ..?'', കൂലം കൂഷമായ എന്റെ ആലോചനയിലൊന്നും .., അങ്ങിനെയൊരു മുഖം തെളിയുന്നില്ല ..!, തെളിയുന്നതെല്ലാം കാശ് അങ്ങോട്ട് കൊടുക്കുവാനുള്ളവരുടെ മുഖങ്ങളാണ് ..!

അത് മനസ്സീന്ന് മാറ്റാന്‍ നോക്കിയാലും .., വീണ്ടും അതെന്നെ തെളിയുന്നു ...!

''ടീ ..., നീ നന്നായിട്ടൊന്ന് ആലോചിച്ചു നോക്ക്യേ ...?, ഇത് കാശ് കൊണ്ട് വരുന്ന ഉറുമ്പ് ആണോ ...?, അതോ കാശ് അങ്ങോട്ട് കൊടുക്കാനുള്ളവര്‍ വന്ന് ഗ്രഹനാഥന്റെ കൊങ്ങക്ക് പിടിക്കാന്‍ പോകുന്നു എന്നുള്ളതിന്റെ മുന്നറിയിപ്പാണോ ...?''

''നിങ്ങളൊന്നു മിണ്ടാതിരിക്ക് മനുഷ്യാ ..., എന്റെ വീട്ടില് .., ഈ ഉറുമ്പ് വന്നാ .., അമ്മ പറയും .., ഇന്ന് കാശ് വരൂന്ന് .., അത് സത്യന്ന്യാ ...!, എന്നൊക്കെ ഈ ഉറുമ്പ് വന്നീട്ടുണ്ടോ .., അന്നൊക്കെ കാശും വന്നീട്ടുണ്ട് ..?''

ഭാര്യയുടെ ആ വിശ്വാസത്തെ മുറിവേല്‍പ്പിക്കാന്‍ .., എനിക്ക് തോന്നിയില്ല .., കാരണം അവര് വല്യ പണക്കാരാണേ ...., ഏലം .., റബ്ബര്‍ .., കുരുമുളക് .., അങ്ങിനെ ഏകജക്കമാന ബിസ്സിനസ്സ്‌കള്‍ ഉണ്ടേ ...!, ഇപ്പോ .., ഉറുമ്പ് വന്നില്ലെങ്കിലും .., അവിടെ കാശ് വരുമെന്ന് ഉറുമ്പിനറിയാം ..., എല്ലാവര്‍ക്കുമറിയാം ...!

അങ്ങിനൊരു ഗമ വെറുതേ ..., തന്റെ മേല് കിട്ടാണേങ്കീ ..., വെറുതേ എന്തിനാ അത് തട്ടിക്കളയണേ .., എന്ന് ഉറുമ്പും വിചാരിച്ചു കാണും .., കാണണം ...!

എന്നാല്‍ എന്റെ സ്ഥിതി അങ്ങിനെയാണോ ...?, ,ഇന്നാരോടാ കാശ് കടം ചോദിക്കേണ്ടന്ന് ആലോചിച്ച് നടക്കണ എന്റെ അടുത്തേക്ക് കാശ് വരൂന്ന് പറയുമ്പോ .., അതിലെന്തോ ഒരു പൊരുത്തക്കേട് ...!

ഇനി ഇപ്പൊ ഈ ഉറുമ്പുകള്‍ .., നമ്മടെ വീടിന്റെ ഉമ്മറത്തൂടെ .., ഷോര്‍ട്ട്കട്ടായിട്ട് അടുത്ത വീട്ടിലെ അവറാച്ചന്റെ അടുത്തേക്കുള്ള പോക്കാണോ ...?

അങ്ങനിപ്പോ .., നമ്മടെ ഓസ് പറ്റി അവറാച്ചന് കാശ് കിട്ടേണ്ടാ ...!

ഈ പഞ്ചാര എടുക്കാന്‍ പോയ ചെക്കനെ കാണാനില്ലല്ലോ ...?, അവന്‍ അത് തിന്ന് തീര്‍ക്കുന്നാ തോന്നണേ ...!

''ടാ ..., ''!, എന്റെ അലര്‍ച്ചയും ...,''ഠിം ..'', ദേ ..., കിടക്കണൂ പഞ്ചാരഭരണി താഴെ ...!

രണ്ട് വായ പോയി .., മൂന്നാമത്തെ വായക്കുള്ള പഞ്ചാരക്കായി കയ്യിട്ടപ്പോഴാണ് എന്റെ വിളി ..!, ആളങ്ങട് ഞെട്ടിപ്പോ .., പഞ്ചാരഭരണി .., ദേ .., താഴെ ...!

ഭാര്യേടെ കത്തുന്ന നോട്ടം എന്റെ മുഖത്ത് ..., ഞാന്‍ കൊടുത്ത ഐഡിയ ആണല്ലോ ..., എന്നെ തിരിഞ്ഞു കൊത്തുന്നത് ...?

ഇനീപ്പോ വിത്തൗട്ട് ചായ തന്നെ ശരണം ...!

പെട്ടെന്നാണ് ആ കൊള്ളിയാന്‍ എന്റെ തലക്കുള്ളില്‍ മിന്നിയത് ...

''ദൈവമേ .., ബംബറ് എടുത്തത് ഇന്നലല്ലേ ...?, അതെങ്ങാനും .., അടിച്ചോ ..?, അതിന്റെ നിമിത്തം കൊണ്ടാണോ .., കാശുറുമ്പുകളുടെ സംസ്ഥാന സമ്മേളനം എന്റെ വീട്ടു മുറ്റത്ത് നടക്കുന്നത് ...?

ആകെ എരിപൊരി സഞ്ചാരം ..., അടിച്ചോ .., ദൈവമേ ...?

പെട്ടെന്നുള്ള എന്റെ സംഭ്രമം കണ്ട് .., ഭാര്യക്ക് പരവേശം ...!, ഇനീപ്പോ കാശ് വരൂന്ന് കേട്ടപ്പോ ഭര്‍ത്താവിന് അറ്റാക്ക് വന്നതാണോ .., അതോ പഞ്ചാര ഭരണി പൊട്ടിയതിലുള്ള വിഷമമോ ...?

''നിങ്ങള് വിഷമിക്കെണ്ടാന്ന്യേ .., പഞ്ചാരഭരണി നമുക്ക് വേറെ വാങ്ങാം ....!''

''നീ .., ഒന്ന് പോടീ കോത്താഴത്തുകാരി ..., പഞ്ചാരഭരണി പൊട്ടെല്യ് .., നിന്റെ തന്തപ്പിടിക്കാണോ വിഷമം ..?ഇത് കാര്യം വേറെ ...''

ഇത് ഞാന്‍ മനസ്സില്‍ പറഞ്ഞതാണേ ..

ഇത് അവളോട് പറഞ്ഞാപ്പിന്നെ .., ലോട്ടറി നോക്കാന്‍ ഞാനുണ്ടാവില്ല ...!

ഏതായാലും ഞാന്‍ പത്രമേടുത്തോണ്ട് പതുക്കെ ഉള്ളിലേക്ക് വലിഞ്ഞു ...,!

ആരും കാണേണ്ടാ ...!

ഇത് നിനക്കെന്നെ .., ബംബറ് .., നിനക്കെന്നെ .., ഉള്ളിലിരുന്നാരോ ധൈര്യം തരുന്നു ...!

മനസ്സാണെങ്കീ .., ഇവിടെയൊന്നും നില്‍ക്കണില്ലാ .., അതങ്ങനെ ..,മാരുതിയില്‍ .., തുടങ്ങി ബെന്‍സ്‌വരേക്കും പോയിവന്നൂ ...!

ഒരു വിധത്തിലാണ് ലോട്ടറി നോക്കാന്‍ തുടങ്ങിയത് .., ആദ്യം അവസാനം തൊട്ട് നോക്കാം .., ഇനീപ്പോ ബംബര് അടിച്ചെന്നു പറഞ്ഞു ഒറ്റയടിക്ക് തട്ടിപ്പോകെണ്ടല്ലോ ...!

കണ്ണുകള്‍ താഴൊട്ട് നിക്കണില്ല .., മേല്‌പ്പോട്ടെക്ക് പറന്നു പോകുന്നു ...!

മേലോട്ട് എത്തും തോറും .., ആവേശം എവര്‍സ്റ്റും താണ്ടി ...!

''ദേ ..., അടിച്ചൂ ..., ഇല്ല ...., വേറെ ആണുങ്ങള്‍ക്ക് അടിച്ചു മോനേ ...!''

''ചായ ഇല്ലേടീ ..., ''!, ആ ദേഷ്യം .., ഭാര്യയോട് തീര്‍ത്തു ...!

''പഞ്ചാര ഇല്ലാണ്ട് .., ചൂടു വെള്ളം അവിടെ ഒരു ഗ്ലാസ്സ് വെച്ചിട്ടുണ്ട് ..., പോയി എടുത്ത് കുടിച്ചോ ...?''

''പഞ്ചാരഭരണി .., ഞാനാണോടീ ..., പൊട്ടിച്ചത് ...?''

''പിന്നെ ..., ഞാനോ ....'?
