<strong>കേരളം കണ്ട ഏറ്റവും വിനാശകാരിയായ വൈറസിന്റെ പേര് എ.കെ.ആന്റണി എന്നാണ്</strong>

വിമോചനസമരം നടന്ന 1957-59 കാലത്ത് ബസ് കത്തിച്ചും പൊതുമുതല്‍ നശിപ്പിച്ചുമാണ് <strong>ആന്റണി</strong> നാടിന്റെ പൊതുമണ്ഡലത്തില്‍ പ്രത്യക്ഷമാകുന്നത്.ഖദറുടപ്പ് തനിയെ കീറി തുന്നലിട്ടും പത്രക്കാരെ അതിരാവിലെ വിളിച്ചുണര്‍ത്തി കൊണ്ടുപോയി അമ്മയുടെ കുഴിമാടത്തില്‍ വീണു പൊട്ടിക്കരഞ്ഞും എന്തെന്ത് അഭ്യാസങ്ങളാണ് ഈ മനുഷ്യന്‍ ചെയ്തുകൂട്ടിയിരിക്കുന്നത്. കേരളം കണ്ട ഏറ്റവും കാപട്യക്കാരനായ രാഷ്ട്രീയക്കാരന്‍ എന്ന് കൂടെയുള്ള കോണ്‍ഗ്രസ് നേതാക്കളാല്‍ വിശേഷിപ്പിക്കപ്പെട്ട ആന്റണി ഈ കൊറോണക്കാലത്തും തല നീട്ടിയിട്ടുണ്ട്.അദ്ദേഹത്തിന്റേതായി വന്ന അഭിമുഖത്തിലെ പ്രസക്ത ഭാഗങ്ങളും വസ്തുകളും :

■ കേരളം നമ്പർ വണ്‍ എന്ന് അഹങ്കരിക്കണ്ട. കൊറോണ ഇനി വേണമെങ്കിലും പടരാം.
( //ആ മനസിന്റെ വൈകൃതം സ്വയം വ്യക്തമാക്കുന്ന വാക്കുകൾ. വിശദീകരണം ആവശ്യമില് // )

■ കേരളത്തിന്റെ ആരോഗ്യ മേഖല മികച്ചതാകാന്‍ കാരണം ഇടതുപക്ഷ സര്‍ക്കാരുകളല്ല. തിരുവിതാംകൂർ രാജാക്കന്മാരാണ്.
( //അതായത് , കണ്ണൂരും കാസര്‍കോഡും ഭരിച്ചിരുന്നത് തിരുവിതാംകൂർ മൂത്ത രാജാവ് കുരിയന്‍ പിള്ള ആയിരുന്നു എന്നാണോ ആന്റണി അര്‍ത്ഥമാക്കുന്നത് ??കന്യാകുമാരിയില്‍ രോഗവ്യാപനം ഉണ്ടെന്നു പറഞ്ഞ് അഭിമുഖത്തില്‍ അദ്ദേഹം മലയാളികളെ ഭീഷണിപ്പെടുത്തുന്നുണ്ട്.
ഇതേ കന്യാകുമാരി ആയിരുന്നല്ലോ തിരുവിതാംകൂർ രാജാക്കന്മാരുടെ പൂണ്ടുവിളയാടല്‍ കേന്ദ്രം ?? അവിടെ പിന്നെന്താ കോവിഡ് വ്യാപിക്കുന്നത് ?? // )

■ തിരുവിതാംകൂർ ആണ് കേരളത്തിൽ ആദ്യമായി ആശുപത്രി തുടങ്ങിയത്. അതുകൊണ്ടാണ് ഇവിടെ മികച്ച ആരോഗ്യ പരിപാലനം ഉണ്ടായത്.
( // ഇന്ത്യയിലെ ആദ്യത്തെ ജനറൽ ഹോസ്പിറ്റൽ 1664ല്‍ ചെന്നൈയിലാണ്. തിരുവനന്തപുരത്ത് 1865ല്‍. ഈ വാദം വച്ചാണെങ്കില്‍ 200 വര്‍ഷം മുമ്പ് ആശുപത്രി തുടങ്ങിയ തമിഴ്നാട് ആകണ്ടേ നമ്പർ വണ്‍ ? //)

■ ആദ്യ വാക്സിന്‍ തിരുവിതാംകൂർ രാജാക്കന്മാര്‍ പരീക്ഷിച്ചു. അതുകൊണ്ടാണ് കേരളത്തിൽ കൊറോണ വ്യാപിക്കാത്തത്.
( //തിരുവനന്തപുരത്ത് വാക്സിന്‍ വന്നത് 1811. മുംബൈയില്‍ 1802. അപ്പോൾ മുംബൈ അല്ലേ കേരളത്തേക്കാള്‍ മുമ്പില്‍ നില്ക്കേണ്ടത് ?// )

ഈ ലോക മണ്ടത്തരങ്ങളൊക്കെ എന്തിനാണ് ആന്റണി പറയുന്നത്?
കേരളമികവിനെ കുറച്ചു കാണിക്കാന്‍ തന്നെ.. അല്ലാതെന്തിന് ??
അതുവഴി ഇടതുപക്ഷ തിളക്കം കുറച്ചെടുക്കാമല്ലോ !!
രാഷ്ട്രീയ എതിരാളികള്‍ക്കു ഗുണമുണ്ടാകുമെന്ന ഭയം മാത്രം ആണിത്.. കുളം കലക്കാന്‍ പെറ്റനാടിനു തന്നെ തീകൊളുത്താന്‍ നോക്കുന്നതിന്റെ പല ഉദാഹരണങ്ങൾ നാം കണ്ടു കഴിഞ്ഞു !!

ഭൂപരിഷ്കരണം, വിദ്യാഭ്യാസ നയം, പൊതു ആരോഗ്യ കേന്ദ്ര ശൃംഖല, സാക്ഷരത, അധികാരവികേന്ദ്രീകരണം, ശാസ്ത്രീയ കാഴ്ചപ്പാട്, വിപ്ലവപ്രസ്ഥാനങ്ങളുടെ പ്രവര്‍ത്തനരീതിയിലൂടെ പകര്‍ന്ന സന്നദ്ധ പ്രവര്‍ത്തന പാരമ്പര്യം, സാമൂഹ്യ ബോധം, രാഷ്ട്രീയ ദിശാബോധം, മികവാര്‍ന്ന നേതൃത്വം എന്നിങ്ങനെ നിരവധി ഗുണങ്ങളുടെ ആകെത്തുകയാണ് കേരളമികവിന്റെ അടിസ്ഥാനം എന്നു മനസിലാക്കാനുള്ള ഭയമാണ് ഇതു പോലുള്ള ലോക ഉഡായിപ്പുകള്‍ പറയാന്‍ ആന്റണിയെ പ്രേരിപ്പിക്കുന്നത്.

ഇതിലെന്തൊക്കെയാണ് രാജകുടുംബമോ കോണ്‍ഗ്രസ് ഭരണമോ നമുക്ക് ഉണ്ടാക്കി തന്നത് ?
ആന്റണി നന്നായതോ ഇല്ലാതായതോ കൊണ്ടല്ല കേരളം മുന്നോട്ടു പോയത്. ആന്റണിയെ ദൂരെ നിര്‍ത്തിയാണ്.അതുപോലെ, കൊറോണ നന്നായിട്ടോ ഇല്ലാതായിട്ടോ നമുക്ക് ജീവിക്കാമെന്നു കരുതണ്ട.
കൊറോണയെ ദൂരെനിര്‍ത്തി നമുക്ക് മുന്നേറണം.
