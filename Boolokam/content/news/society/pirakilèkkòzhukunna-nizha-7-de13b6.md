ടി.വി. കാണുന്നതിലുപരി ..,അവരുടെ വീട്ടിലേക്കുള്ള യാത്രകളില്‍ , എന്നെ ആകര്‍ഷിച്ച മുഖ്യ ഘടകം ..., എന്റെ മനസ്സ് അടിയറവ് പറഞ്ഞുപോയ ആ സാന്നിധ്യം തന്നെയായിരുന്നു ....ടി.വി. കാണുമ്പോള്‍ ..., എല്ലായിപ്പോഴും ..., അവരുടെ സമീപത്തുതന്നെ ഇരിക്കുവാന്‍ ഞാന്‍ പ്രത്യേകം ശ്രദ്ധിക്കുമായിരുന്നു ....!

ട്രൌസര്‍ ധാരിയായ എനിക്ക് ഒരു ചെറിയ കുട്ടിയുടെ പ്രതിച്ചായ ഉള്ളതിനാല്‍ .., പ്രമിളയോടുള്ള എന്റെ കൂടുതല്‍ അടുപ്പം .., ആരും സംശയത്തോടെ നോക്കി കണ്ടിരുന്നില്ല ...., ഇതെനിക്ക് കൂടുതല്‍ സ്വാതന്ത്ര്യം ലഭിക്കുന്നതിന് ഇടയാക്കി ....!

പലപ്പോഴും ..., പ്രമിളയുടെ എന്നോടുള്ള പെരുമാറ്റം കാണുമ്പോള്‍ ..., എന്നോടെന്തോ .., ഒരു പ്രത്യേക തരത്തിലുള്ള കൂടുതല്‍ അടുപ്പം അവര്‍ പ്രകടിപ്പിച്ചിരുന്നു ..., ചോക്ലേറ്റുകളും ..., പേനകളും ...,മറ്റും എനിക്ക് സമ്മാനമായിതരുക ..., പോക്കറ്റു മണി തരുക .., അവരുടെ മനസ്സില്ലുള്ള വിഷമങ്ങളും ..., ദുഖങ്ങളും ....എന്നോട് തുറന്നു പറഞ്ഞ് ..., അതില്‍ ആശ്വാസം കണ്ടെത്തുവാന്‍ ശ്രമിക്കുക ....!, ഞാന്‍ എന്തെങ്കിലും നിസ്സാര തെറ്റുകള്‍ ചെയ്താല്‍ ..., എന്നെ സ്‌നേഹപൂര്‍വ്വം ശാസിക്കുക ..., ചെവി പിടിച്ചു തിരുമ്മുക ...., ഇങ്ങനെ വളരെയധികം സ്വാതന്ത്ര്യം അവര്‍ എന്നില്‍ പ്രകടിപ്പിച്ചിരുന്നു ....!

വേദനിപ്പിച്ചതിനു ശേഷം ..., ആശ്വസിപ്പിക്കുന്നതിനായി എന്നെ കെട്ടിപിടിക്കുമായിരുന്നു ..., അപ്പോള്‍ ആ കൈകള്‍ക്ക് കരുത്ത് അല്പം കൂടുന്നില്ലേ ..., എന്നെനിക്ക് സംശയം തോന്നും ...!എന്നാല്‍ അത് സംശയം മാത്രമല്ല സത്യം തന്നെ ആയിരുന്നു .., പറഞ്ഞറിയിക്കാനാകാത്ത വേറേതോ വികാരത്തിന്റെ ഊഷ്മളത ആയിരുന്നു ..., ആ ... ആശ്ലേഷങ്ങള്‍ക്ക് ഉണ്ടായിരുന്നത് ....

ഈ സമയം .., എന്റെ മുഖം .., അവരുടെ മാറില്‍ അമര്‍ന്നു ചേരുമായിരുന്നു .., അതെനിക്ക് അസുലഭമായ ഒരു അനുഭൂതി പ്രധാനം ചെയ്തിരുന്നു .., അവരില്‍ നിന്നും ഉയര്‍ന്ന സ്ത്രീ ശരിരത്തിന്റെ നൈസര്‍ഗ്ഗീക ഗന്ധം .., എന്നിലെ വികാരത്തെ ഉത്തുംഗശ്രിംഗങ്ങളില്‍ എത്തിക്കും ..., മാറില്‍ തലചായിച്ചു നില്‍ക്കുന്ന , എന്റെ ശിരസ്സിനു പിന്നില്‍ ..., ആശ്വസിപ്പിക്കാനെന്നവണ്ണം അവര്‍ , അമര്‍ത്തി തട്ടുമായിരുന്നു ...!

ഈ തരത്തിലുള്ള പരിലാളനങ്ങള്‍ .., കൌമാരത്തിലുള്ള എന്റെ ലൈംഗിക അഭിലാഷങ്ങളെ പരകോടിയിലെത്തിക്കുമായിരുന്നു ..., അടിക്കടി എന്റെ വീട്ടിലേക്കുള്ള അവരുടെ വരവും ..., എന്നോടൊത്തു കൂടുതല്‍ സമയം ചെലവഴിക്കാനുള്ള ശ്രമവും ..., എല്ലാം ...., എന്നോടുള്ള ആസക്തിയുടെ പ്രതിഫലനങ്ങളായി ഞാന്‍ കണ്ടു ...!

ഈ അസുലഭ നിമിഷങ്ങളെ ഒന്നുകൂടി അടുത്തറിയുവാന്‍ ഉതകുന്നതായിത്തീര്‍ന്നു ..., എന്റെ പാഠ വിഷയങ്ങളില്‍ .., അവര്‍ എന്നെ സഹായിച്ചു വന്നത് ...!പഠിക്കാനായുള്ള .., ഈ സമയങ്ങളില്‍ .., ഞാന്‍ പഠിച്ചത് .., പാഠപുസ്തകങ്ങള്‍ ആയിരുന്നില്ല ...,മറിച്ചു ..;ജീവിതത്തില്‍ ഒരിക്കലും മറക്കാനാകാത്ത ധാരാളം ഏടുകള്‍ എനിക്കത് സമ്മാനിച്ചു ...!

കൗമാര സ്വപ്നങ്ങളില്‍ എപ്പോഴും താലോലിക്കാന്‍ ഇഷ്ടപെടുന്ന ലൈംഗികതയുടെ മാസ്മരിക ലോകത്തിലേക്കുള്ള ഒരു വാതായനമായിരുന്നുവത് ....!വികാരത്തിന്റെ മേച്ചില്‍പുറങ്ങളിലൂടെ കുതിച്ചു പായുന്ന മനസ്സിന്റെ .., മനോമുകുരങ്ങളില്‍ തെളിഞ്ഞു വരുന്ന പല ജിത്‌നാസകളുടെയും ..., അനുഭൂതികളുടെയും .., അനേകായിരം ചോദ്യങ്ങളുടെയും ..., ഉത്തരങ്ങളായിരുന്നു ..., ആ ...., സായാഹ്നങ്ങള്‍ ...!

സ്വപ്നങ്ങളില്‍ നിന്ന് യാതാര്‍ത്ഥ്യത്തിലേക്കുള്ള ..., കൌമാരക്കാരന്റെ പ്രയാണം ..., ഞരമ്പുകള്‍ പൊട്ടിത്തെറിക്കുന്നത്രയും ശക്തമായ വികാരമൂര്‍ച്ച ..., ശരിരം മുഴുവനും .., കൂലം കുത്തിയൊഴുകുന്ന രക്തത്തിന്റെ പ്രളയ പാച്ചില്‍ ..., പറഞ്ഞറിയിക്കാനാകാത്ത വികാരങ്ങളുടെ വിസ്‌ഫോടനം ..., ശരിരത്തിന്റെ എറ്റവും ശക്തിമത്തായ കാലഘട്ടം ..., എല്ലാം അറിയുവാനുള്ള തൃഷ്ണ ..., അറിയുംതോറും കൂടുതല്‍ ആസക്തി .....!

കടിഞ്ഞാണില്ലാതെ കുതിച്ചുപായുന്ന യൌവ്വനത്തിന്റെ തിളപ്പും ..., ഭര്‍ത്താവിന്റെ ..., അകല്‍ച്ചയും .., അവരില്‍ ഉണര്‍ത്തിയിരുന്ന വികാരത്തിന്റെ അളവ് വളരെ വലുതായിരുന്നു ...!

കുറച്ചു ദിവസങ്ങള്‍ മാത്രം അനുഭവഭേദ്യം ആയ ഭര്‍ത്താവിന്റെ സാമീപ്യം ...; ആ വികാരങ്ങളെ പിന്നീട് കടിച്ചമര്‍ത്തപ്പെടെണ്ടി വന്നപ്പോള്‍ ...; സാധാരണ പോലെ ഏതൊരാള്‍ക്കും ഉണ്ടാകാവുന്ന ഒരു സെടക്ഷന്‍ തന്നെയായിരുന്നു അവര്‍ക്കും ...!

കൂടുപോട്ടിച്ചു പുറത്തുചാടാന്‍ വെമ്പി നില്‍ക്കുന്ന വികാരത്തിന്റെ ഒഴുക്കിന് .., എന്റെ സാമീപ്യം ഒരു കാരണമായിത്തീര്‍ന്നിരിക്കാം ...., അണ തകര്‍ക്കാന്‍ വെമ്പി നില്‍ക്കുന്ന ജലാശയത്തിന്റെ ഷട്ടര്‍ തുറന്ന് .., അതിന്റെ ശക്തമായ മര്‍ദ്ദം അല്പം കുറക്കപ്പെടുന്നതിനു തുല്യമായിരുന്നുവത് .....!

പൂര്‍ണ്ണമായും മതില്‍ക്കെട്ടുകള്‍ ഭേദിക്കാതെ ..., ആഗ്രഹത്തിന്റെ സഫലികരണത്തെയും ..., മനസ്സിന്റെ കുറ്റബോധത്തെയും ...., ഒഴിവാക്കനാകാതെ .., രണ്ടിന്റെയും മേല്‍ വിദഗ്ധമായൊരു മൂടുപടം സമ്മാനിച്ചുകൊണ്ട് ...; എന്നോ അനുഭവിച്ച ആ സുഖകരമായ ലൈംഗികതയുടെ , പൂര്‍ണ്ണമായ തനിയാവര്‍ത്തനത്തെ ഒഴിവാക്കി .., ഭാഗികമായി സംതൃപ്തിയടയുന്ന ഒരു ഗെയിം പ്ലാനിങ്ങ് ആയിരുന്നുവത് ...! അതോ എന്നിലെ കൌമാരത്തിന് അവരെ തൃപ്തിപെടുത്താന്‍ ആകില്ലെന്ന തെറ്റിദ്ധാരണയോ ....? അതോ ...പൂര്‍ണ്ണമായും താന്‍ തെറ്റു ചെയിതിട്ടില്ലെന്ന് സ്വയം മനസ്സിനെ ബോധിപ്പിക്കുവാനുള്ള ..., മനസ്സിന്റെ തന്നെ ഒരു മറുവിദ്യയോ ....!

എന്തൊക്കെത്തന്നെയായാലും ..., ഞാന്‍ ആദ്യമായി .., സെക്‌സിന്റെ ആ ..., വലിയ കാന്‍വാസിലേക്ക് പിച്ചവെച്ചത് പ്രമിളയില്‍കൂടിയായിരുന്നു ....!
പഠനത്തെക്കാള്‍ ഉപരി ഞങ്ങള്‍ ചര്‍ച്ച ചെയിത വിഷയങ്ങള്‍ മറ്റു പലതുമായിരുന്നു ...! സംഭാക്ഷണത്തിന്റെ ആരംഭം എന്തുതന്നെ ആയാലും ..., അവസാനം അത് ചെന്നെത്തി നില്‍ക്കുന്നത് ലൈംഗികതയിലായിരുന്നു...!

മുന വെച്ചുള്ള എന്റെ ചില ചോദ്യങ്ങളും ..., അതിനുള്ള അവരുടെ ഉത്തരങ്ങളും .., ഞങ്ങളെ ഏറെ വികാരഭരിതരാക്കിയിരുന്നു ...., സംസാരത്തിലുടെയുള്ള ആനന്ദം പതുക്കെ ..., പതുക്കെ ..,സ്പര്‍ശനത്തിലൂടെ പുരോഗമിക്കുന്നത് ..., ഞങ്ങള്‍ രണ്ടുപേരും ഒരുപോലെ രസിച്ചു കൊണ്ടായിരുന്നു ...!

സ്വൊതവെ ഒരു നാണം കുണുങ്ങി ആയ എന്റെ ..., നാണം ..., ഈ സമയം വിശദീകരിക്കാന്‍ പറ്റാത്ത അത്ര ആയിരുന്നു .....!സുഗമമായി കേളികൊട്ടിയാടിയിരുന്ന ..., ഈ നാടകത്തിന് .., തിരശ്ശില വീഴുന്നത് .., അവരുടെ ഭര്‍ത്താവിന്റെ അടുത്തേക്കുള്ള അവരുടെ ആകസ്മിക യാത്രയായിരുന്നു ....!
