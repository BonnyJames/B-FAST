കുറച്ചു നാള്‍ മുന്‍പ് ഞാന്‍ എഴുതിയ ഒരു <a href="https://wordpress-972788-3403151.cloudwaysapps.com/archives/60745" target="_blank" rel="noopener">ലേഖനത്തില്‍</a> സീക്രട്ട് ഫയലുകള്‍ എങ്ങനെ ഒരു ഇമേജ് ഫയല്നു പിന്നില്‍ ഹൈഡ് ചെയ്യാം എന്ന് കാണിച്ചിരുന്നു. നിങ്ങള്‍ ആ ലേഖനം വായിച്ചു കാണും എന്ന്‌ കരുതുന്നു. നമ്മുടെ സീക്രട്ട് ഫയലുകള്‍ അടങ്ങുന്ന ഒരു ഫോള്‍ഡര്‍ എങ്ങനെ ആരും കാണാതെ ഹൈഡ് ചെയ്യാം എന്ന് ഈ ലേഖനത്തില്‍ പരിചയപ്പെടാം. ഒരു പക്ഷെ ഇത് വായിക്കുന്ന ചിലര്‍ക്കെങ്കിലും ഇത് അറിയാമായിരിക്കും. അങ്ങനെയ ഉള്ളവര്‍ ദയവു ചെയ്തു ക്ഷമിക്കുക. :)

ആദ്യമായി ഫോള്‍ഡര്‍ C ഡ്രൈവ് ഓപ്പണ്‍ ചെയ്യുക. C ഡ്രൈവില്‍ users എന്നാ ഫോള്‍ഡര്‍ കാണാം. അത് ഓപ്പണ്‍ ചെയ്യുക. അതില്‍ നിങ്ങള്‍ ഇപ്പോള്‍ ലോഗിന്‍ ചെയ്തിരിക്കുന്ന username ല്‍ ഒരു ഫോള്‍ഡര്‍ കാണാം. നിങ്ങള്‍ക്ക് ഹൈഡ് ചെയ്യേണ്ട ഫോള്‍ഡര്‍ കോപ്പി ചെയ്തു ഈ user folder ല്‍ പേസ്റ്റ് ചെയ്യുക.

ഇനി കമാന്‍ഡ് പ്രോംപ്റ്റ് ഓപ്പണ്‍ ചെയ്യുക. കമാന്‍ഡ് പ്രോംറ്റില്‍ താഴെ പറയുന്ന കമാന്‍ഡ് ടൈപ്പ് ചെയ്യുക.

(windows7 ആണെങ്കില്‍ start--&gt;all programs--&gt;accessories--&gt;command prompt

windows XP ആണെങ്കില്‍ start--&gt;programs--&gt;accessories--&gt;command prompt)

( ശ്രദ്ധിക്കുക ഇവിടെ my folder എന്നുള്ളത് ഞാന്‍ ഹൈഡ് ചെയ്യാന്‍ ഉദ്ദേശിക്കുന്ന ഫോള്‍ഡര്‍ ആണ്. )

<strong>attrib +h +s "myfolder"</strong>

ഇവിടെ " " ന്‍റെ ഉള്ളില്‍ കൊടുക്കേണ്ടത് നിങ്ങള്‍ക് ഹൈഡ് ചെയ്യേണ്ട ഫോള്‍ഡറിന്റെ നെയിം ആണ്.

ഈ കമാന്‍ഡ് ടൈപ്പ് ചെയ്തു എന്റര്‍ കീ പ്രെസ്സ് ചെയ്യുക. ഇനി കമാന്‍ഡ് പ്രോംപ്ടിനു പുറത്തു എവിടെയെങ്കിലും റൈറ്റ് ക്ലിക്ക് ചെയ്തു റിഫ്രെഷ് ചെയ്തു നോക്കൂ!! <strong>BOoom !!</strong> നിങ്ങളുടെ ഫോള്‍ഡര്‍ അപ്രത്യക്ഷമായതായി കാണാം.

ഇനി ഹൈഡ് ചെയ്ത ഫോള്‍ഡര്‍ എങ്ങനെ ഓപ്പണ്‍ ചെയ്യാം എന്ന് നോക്കാം.വീണ്ടും കമാന്‍ഡ് പ്രോംപ്റ്റ് ഓപ്പണ്‍ ചെയ്യുക. ഇനി താഴെ പറയുന്ന കമാന്‍ഡ് ടൈപ്പ് ചെയ്യുക.
