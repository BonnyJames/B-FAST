<strong>രാഗീത് ആർ ബാലൻ</strong>

കുഞ്ഞു മറിയ ❣️

കുഞ്ഞു മറിയ : എടിയേ..ത്രേസ്യമോ..ആരും ഇല്ലേ ഇവിടെ.?. എടി ത്രേസ്യമോ..എവിടെ പോയി.. ത്രേസ്യമേ.. പൂയ്...

ജോയ് : ഐ ഹാവ് ടു ഗെറ്റ് ബാക്ക് ടു ബാംഗ്ലൂർ ആസ് സൂൺ ആസ് പോസ്സിബിൾ.. ഐ ഹേറ്റ് ദിസ്‌ പ്ലേസ്.

കുഞ്ഞു മറിയ : ത്രേസ്യമേ...

ജോയ് : ആരാ ഈ കിടന്നു അലറുന്നെ.. ഒന്ന് പോയി നോക്കിയേ

എടി കൊച്ചു ത്രേസ്...ഇവരൊക്കെ ആരാടാ ജോയെ

ജോയ് : റീത്തയുടെ ചേച്ചിയും ഭർത്താവും മക്കളുമൊക്കെയാ.
(കുഞ്ഞു മറിയ അതിഥികളിൽ ഒരാളായ കുട്ടിയുടെ അടുത്ത് ചിരിച്ചു കൊണ്ട് ചെന്ന് ചോദിക്കുന്നു )

കുഞ്ഞു മറിയ : നിന്റെ പേര് എന്നതാടി കൊച്ചേ

കുട്ടി : മറിയ ഫ്രാൻസിസ്

കുഞ്ഞു മറിയ : എന്റെ ഇശോ യെ എന്റെ പേരും മറിയെന്നാ..കുഞ്ഞു മറിയ

കൊച്ചു ത്രേസ്യ : അല്ല ഇതാരാ

കുഞ്ഞു മറിയ : വിരുന്നുകാരൊക്കെ വന്നിരിക്കുമ്പോ നീ എന്നതാടി പേരക്കകത്തു കയറി ഒളിച്ചു ഇരിക്കിന്നെ?ഓ ഊണിനുള്ള വട്ടം കൂട്ടുമായിരിക്കും.. കോഴിയോ താറാവോ?

കൊച്ചു ത്രേസ്യ : പോടി.. നീ ഇങ്ങു വന്നേ

കുഞ്ഞു മറിയ : ഞങ്ങള് പഴയ കൂട്ടുകാരാ. പണ്ട് ഒരുമിച്ചു പാടത്തു കൊയ്യാൻ പോകുന്ന കാലം തൊട്ടേ ഉള്ള കൂട്ടാ. ഒറ്റ കെട്ടായിരുന്നു. പാടത്തു പണിയൊക്കെ ഞാനാ പഠിപ്പിച്ചേ ഈ മടിച്ചി കോതയെ.

കൊച്ചു ത്രേസ്യ : നീ വാടി
കുഞ്ഞു മറിയ : വരുന്നെടി.. ഇതു എന്നതൊക്കെയാ പലഹാരങ്ങള്
(അതിഥികൾക്കായി കൊണ്ട് വെച്ച പലഹാരങ്ങൾ നോക്കി പാത്രത്തിൽ നിന്നും എടുത്തു രൂചിച്ചു നോക്കുന്നു )മ്മ് ഒരു ഇച്ചിരി കാറ ചൊവയുണ്ട് ട്ടാ..
(കയ്യിൽ ഉള്ള സഞ്ചിയിൽ നിന്നും പലഹാരങ്ങൾ എടുത്തു പാത്രത്തിൽ ഉള്ളവ നീക്കി പകരം അവ വെക്കുന്നു )

ദാണ്ടേ ഇതൊന്നു തിന്നു നോക്കിയട്ടെ.. വട്ടയപ്പം..മുറുക്ക്.. ഉണ്ണി അപ്പം.. ദാ തിന്നട്ടെ. ഇതേ ബേക്കറിയിൽ നിന്നും വാങ്ങിച്ചതൊന്നും അല്ല. ഞാൻ എന്റെ സ്വന്തം കൈ കൊണ്ട് മാവ് കുഴച്ചു ഉണ്ടാക്കിയതാ
കൊച്ചു ത്രേസ്യ : നീ വാടി ഇങ്ങോട്ട്

കുഞ്ഞു മറിയ : ഹായ് മക്കളെ ദേ മക്കൾക്ക്‌ വേണ്ടി വല്യമ്മച്ചി ഉണ്ടാക്കികൊണ്ട് വന്നതാ.. തിന്നോ തിന്നോ
( നടന്നു അകത്തേക്ക് പ്രവേശിക്കുമ്പോൾ കുട്ടികൾ കളിക്കുന്നത് കണ്ടു കൈയിലുള്ള സഞ്ചിയിൽ നിന്നും പലഹാരങ്ങൾ എടുത്തു വീതിച്ചു നൽകുന്നു )
ഇന്നലെ ഇങ്ങോട്ട് വരാൻ ഒരുങ്ങിയതാടി.കൊച്ചുങ്ങളൊക്കെ അവധിക്കു വന്നേക്കുവല്യോ അതുങ്ങൾക്ക് എന്തെങ്കിലും തിന്നാൻ ഉണ്ടാക്കിക്കൊണ്ട് വരേണ്ടയോ എന്ന് കരുതിയ ഇന്നേക്ക് ആക്കിയത്.

കൊച്ചു ത്രേസ്യ : മതി വാടി.. തോമസ് കുട്ടി ഈ മാസം അമേരിക്കെന്നു വരുന്നു പറഞ്ഞിട്ട് വിവരം ഒന്നും ഇല്ല്യോടി

കുഞ്ഞുമറിയ :മ്മ് അവന്റെ വിവരം അറിയണമെങ്കിൽ അങ്ങ് എൽസയുടെ അങ്ങ് പോകേണ്ടയോ.. കെട്ട്യോടെ വീട്ടിൽ അല്ലെ അവൻ ഫോൺ വെച്ചു കൊടുത്തേക്കുന്നത്. അവർക്കു ആണെങ്കിൽ ഞാൻ അങ്ങോട്ട്‌ ചെല്ലുന്നതെ ഇഷ്ടമല്ല.

കൊച്ചുത്രേസ്യ : ഓ...

കുഞ്ഞുമറിയ : അല്ല അതിനവനെ പറഞ്ഞ മതി. അമേരിക്കയിൽ നിന്ന് വല്ലപ്പോഴും വരുമ്പോഴേ അവൻ അവിടെ പോയല്ലേ താമസം.. നമ്മുടെ വീട്ടിൽ മുഴുവൻ കൊതുകാണെന്നു.ആയിക്കോട്ടെ ഞാൻ ആയിട്ടൊന്നും എതിര് പറയാൻ നിന്നില്ല.. ഇന്നലെ ഞാൻ അവിടെ പോയിരുന്നു. വരുന്ന വിവരം ഒന്നും അറിയിച്ചിട്ടില്ലെന്നു.
(വീടിന് പുറത്തു ഇതേ സമയം റീത്ത കുട്ടികൾ കഴിച്ചു കൊണ്ടിരുന്ന പലഹാരങ്ങൾ അവരുടെ കയ്യിൽ നിന്നും വാങ്ങി മണ്ണിലേക്ക് വലിച്ചു എറിഞ്ഞു കുട്ടികളെ വഴക്ക് പറയുന്നു )

റീത്ത : കളയെടി അത്..കളയാൻ അല്ലെ പറഞ്ഞത്.. നിങ്ങൾക്ക് തിന്നാൻ ഉള്ളത് ഇവിടില്ലേ?വൃത്തിയില്ലത്ത ഓരോന്ന് കൊച്ചുങ്ങൾ തിന്നുമ്പോൾ പറഞ്ഞു കൊടുക്കണ്ട്‌ നീയും കൂട്ടത്തിൽ അങ്ങ് കൂടുവാണോടാ.നാശങ്ങള്.. ആര് എന്ത് കൊണ്ടുവന്നാലും ഒന്നും കണ്ടിട്ടില്ലാത്ത പോലെ കൈ നീട്ടി അങ്ങ് വാങ്ങിക്കോളും.വിയർപ്പിലും ചെളിയിലും കിടക്കുന്ന കൂട്ടമാ. കണ്ണി കണ്ട വൃത്തികേടുകൾ ഒക്കെ കാണും അതിൽ. തിന്നു വല്ല അസുഖം വന്നാൽ.. കളയത് (കുട്ടിയുടെ കയ്യിൽ ഉണ്ടായിരുന്ന വട്ടയപ്പം മുറ്റത്തേക്ക് വലിച്ചു എറിയുന്നു )

ഇതു കണ്ടു കൊണ്ട് ഇരുന്ന കുഞ്ഞു മറിയ കൊച്ചു ത്രേസ്യയോട്
കുഞ്ഞു മറിയ : രാത്രി ഒരു പോള കണ്ണടക്കാതെ ഇരുന്നു ഉണ്ടാക്കിയതാ..
(ഇതും പറഞ്ഞു വിതുമ്പി കൊണ്ട് കുഞ്ഞു മറിയ ആ വീട്ടിൽ നിന്ന് ഇറങ്ങുന്നു.)
ആ രംഗത്തിനൊപ്പം വരുന്ന പശ്ചാത്തല സംഗീതവും കൂടെ ആകുമ്പോൾ കാണുന്ന ഏതൊരു പ്രേക്ഷകന്റെയും കണ്ണൊന്നു നിറയും. തീർച്ച.

ചില സിനിമകളിലെ ചില രംഗങ്ങളും ചില കഥാപാത്രങ്ങളും മതിയാകും നമുക്ക് എന്നെന്നും ആ സിനിമ ഓർത്തിരിക്കാൻ.അങ്ങനെയുള്ള രംഗങ്ങളും കഥാപാത്രങ്ങളും കാണുന്ന പ്രേക്ഷകന്റെ മനസ്സിൽ ആഴത്തിൽ പതിഞ്ഞു പോകുന്നവയാണ്.എന്നിലെ പ്രേക്ഷനോട് അത്രയേറെ വൈകാരികമായി അടുത്ത് നിൽക്കുന്നതും ആഴത്തിൽ പതിഞ്ഞു പോയതുമായ ഒന്നാണ് കുഞ്ഞു മറിയയും ആ ഒരു രംഗവും.

എത്ര മനോഹരമായിട്ടാണ് കെ പി എ സി ലളിത എന്ന അതുല്യ പ്രതിഭ കുഞ്ഞു മറിയെ അവതരിപ്പിച്ചിരിക്കുന്നത്.

"രാത്രി ഒരു പോള കണ്ണടക്കാതെ ഇരുന്നു ഉണ്ടാക്കിയതാ" എന്ന് പറഞ്ഞു നിസഹായതയോടെ വിതുമ്പി വീടിനു പടിയിറങ്ങി കുഞ്ഞു മറിയ പോകുമ്പോൾ..... എനിക്ക് അടുത്തറിയാവുന്ന ആരോ ഒരാൾ എനിക്ക് പ്രിയപ്പെട്ട ഒരാൾ പോകുന്ന പോലെയാണ് തോന്നിയിട്ടുള്ളത്.....
