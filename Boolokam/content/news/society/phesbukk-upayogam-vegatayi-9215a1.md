മാസം 1.28 ബില്ല്യന്‍ ആക്ടീവ് യൂസര്‍മാരോടെ ഫേസ്ബുക്ക് എന്ന ലോകത്തിലെ നമ്പര്‍ വണ്‍ സോഷ്യല്‍ നെറ്റ് വര്‍ക്ക് ലോകത്തിനു മുന്‍പില്‍ തലയുയര്‍ത്തി നില്ക്കാന്‍ തുടങ്ങിയിട്ട് കുറച്ചു കൊല്ലങ്ങളായി. വാട്ട്സ് ആപ്പും ഫേസ്ബുക്ക് മെസഞ്ചറും വന്നെങ്കിലും സിസ്റ്റം വഴി ഫേസ്ബുക്ക് ഉപയോഗിക്കുന്നവരുടെ എണ്ണം കുത്തനെ കൂടുക തന്നെയാണ്. അങ്ങിനെ ഉള്ളവര്‍ക്കായി തങ്ങളുടെ ഫേസ്ബുക്ക് ഉപയോഗം അതിവേഗത്തില്‍ സാധ്യമാക്കുന്നതിന് വേണ്ടി ഫേസ്ബുക്ക് ഒരുക്കിയിരിക്കുന്ന ചില കീവേര്‍ഡ്‌ ഷോര്‍ട്ട് കട്ടുകള്‍ പരിചയപ്പെടുത്തുകയാണ് ഈ പോസ്റ്റിലൂടെ.

ഒരു കാര്യം പ്രത്യേകം ശ്രദ്ധിക്കേണ്ടത് ബ്രൌസര്‍ മാറുമ്പോഴും ഓപ്പറേറ്റിംഗ് സിസ്റ്റം മാറുമ്പോഴും കീവേര്‍ഡ്‌സ് ഉപയോഗിക്കേണ്ട രീതിയില്‍ മാറ്റം വരും എന്നുള്ളതാണ്. അത് കൊണ്ട് തന്നെ കീവേര്‍ഡ്‌ കാണും മുന്‍പേ ആ കാര്യങ്ങള്‍ താഴെ വായിക്കാം. താഴെ കൊടുത്തവയില്‍ ഹാഷ് ടാഗ് (#) ഉള്ളിടത്ത് അതാത് കീവേര്‍ഡ്‌ നല്‍കിയാല്‍ ആ ഷോര്‍ട്ട് കട്ട് വര്‍ക്കാകും.

<strong>Internet Explorer for PC: Alt + #, then Enter
Firefox for PC: Shift + Alt + #
Safari for Mac: Ctrl + Opt + #
Firefox for Mac: Ctrl + Opt + #
Chrome for Mac: Ctrl + Opt + #
Chrome for PC: Ctrl + #</strong>

ഹാഷ് ടാഗിന് (#) പകരം ഉപയോഗിക്കേണ്ട കീവേര്‍ഡുകള്‍

<strong>0 - Help
1 - Home
2 - Timeline
3 - Friends
4 - Inbox
5 – Notifications
6 – Settings
7 - Activity Log
8 - About
9 - Terms</strong>

ന്യൂസ്‌ ഫീഡില്‍ ഉപയോഗിക്കാവുന്ന ഷോര്‍ട്ട് കട്ടുകള്‍ (ഇതില്‍ താഴെ നല്‍കുന്ന കീവേര്‍ഡ്‌ മാത്രം ഉപയോഗിച്ചാല്‍ മതി. # ഉപയോഗിക്കേണ്ടതില്ല)

<strong>j and k - Scroll between News Feed stories
p - Post a new status
l - Like or unlike the selected story
c - Comment on the selected story
s - Share the selected story
o - Open attachment of the selected story
/ - Search
q - Search chat contacts
? - Open the list of keyboard shortcuts while in News Feed</strong>

ഇനി ഡെസ്ക്ടോപ്പ് വേര്‍ഷനിലെ വെബ്‌ മെസഞ്ചറില്‍ ഉപയോഗിക്കാവുന്ന ഷോര്‍ട്ട് കട്ടുകള്‍

<strong>CTRL + g - Search conversations
CTRL + q - Show/hide keyboard shortcuts
CTRL + Delete - Archive/unarchive conversation
CTRL + j - Mark as spam
CTRL + m - Start a new message
CTRL + i - Go to Inbox
CTRL + u - Go to Other</strong>

മുകളില്‍ നല്‍കിയ ഷോര്‍ട്ട് കട്ടുകള്‍ കാണാതെ പഠിച്ചു നോക്കൂ. പിന്നെ ഫേസ്ബുക്ക് ഉപയോഗം വളരെ എളുപ്പമാകും നിങ്ങള്‍ക്ക്.
