ഇനി മൊബൈലില്‍ മലയാളം വായിക്കാം

മൊബൈലില്‍ മലയാളം വായിക്കാന്‍ ചെയ്യേണ്ടത് ഇത്രമാത്രം.

അദ്യം http//:m.opera.com എന്ന website നിങ്ങളുടെ മോബൈലിലെ വെബ് ബ്രൊസ്സറില്‍ open ചെയ്യുക.
പിന്നീട് അതില്‍നിന്നും Opera Mini Browser നിങ്ങളുടെ മോബൈലില്‍ install ചെയ്യുക.

ഇനി Opera Mini Browser open ചെയ്ത് Adress Bar ല്‍ config: എന്ന് ടൈപ്പ് ചെയ്യുക.

അപ്പോള്‍ അവിടെ Power User Settings പേജ് ഓപ്പണ്‍ ആയി വരും.അതിന്റെ ഏറ്റവും താഴെ Use bitmap fonts for complex scripts എന്ന option കാണും.

അവിടെ NO എന്ന് കാണുന്നത് YES എന്ന് മാറ്റി സേവ് ചെയ്യുക.

ഇനി Opera Mini യില്‍ നിങ്ങള്‍ക്ക് മലയാളം വായിച്ചു തുടങ്ങാം.
