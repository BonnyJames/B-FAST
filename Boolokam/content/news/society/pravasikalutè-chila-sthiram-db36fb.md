അന്യ നാട്ടില്‍ പോയി സ്വന്തം കുടുബത്തിനു വേണ്ടി എല്ല് മുറിയെ പണിയെടുക്കുന്നവരാണ് പ്രവാസികള്‍.!

ഈ പ്രവാസികളുടെ വിഷമങ്ങളും വേദനകളും സ്വപ്നങ്ങളുമെല്ലാം അവരുടെ ചില വാക്കുകളില്‍, വചനങ്ങളില്‍ ഒളിഞ്ഞു ഇരിക്കുനുണ്ടാകും..

ഇതാ ചില പ്രവാസി വചനങ്ങള്‍...

1. ' കടങ്ങള്‍ വീടിയിട്ടൊന്നും നാട്ടില്‍ പോക്ക് നടക്കില്ല ..! '

2. ' പെണ്ണ് കെട്ടിയാല്‍ പ്രവാസത്തില്‍ നിന്നൊരു മോചനം പിന്നെ ഉറപ്പിക്കണ്ട '

3. ' വിസയുടെ കാഷ് ഒരിക്കലും നമ്മള്‍ക്ക് മൊതലായി തോന്നൂലാ '

4. ' കുടുംബക്കാര്‍ക്കൊന്നും ഒരിക്കലും നമ്മുടെ സാലറിയുടെ കണക്ക് പറഞ്ഞ് കൊടുക്കരുത് '

5. ' ആരോടായാലും ഗള്‍ഫിലെ ഉള്ള സ്ഥിതി തുറന്ന്! പറയുക ഗമക്ക് ഇല്ലാത്ത കാര്യങ്ങള്‍ പറഞ്ഞാല്‍ അനുഭവിക്കും '

6. ' കയ്യഴഞ്ഞ് കൊടുക്കുമ്പോഴും ഒരു നുള്ള് അവനവന് വേണ്ടി മാറ്റി വെക്കുക '

7. ' വീടാവാതെ നിര്‍ത്തി പോകുന്നത് മനപ്രയാസം കൂട്ടും '

8. ' കുട്ടികളെയും ഭാര്യയേയും എല്ലാം ഗള്‍ഫിലെ പൈസ ഉണ്ടാക്കുന്നത് എങ്ങനെയാണെന്ന് പറഞ്ഞ് കൊടുത്ത് മനസിലാക്കുക '

9. ' പരമാവധി വിസ കച്ചോടം ഒഴിവാക്കുക '

10. ' കാഷ് കൊണ്ട് മുന്നോട്ട് പോകുന്ന കൂട്ട് കെട്ടുകള്‍ ഗള്‍ഫില്‍ ഒഴിവാക്കുക '

ഇങ്ങനെ കുറേയുണ്ട് ... ബാക്കി വൈകാതെ പറയാം ...!
ഇതൊക്കെ മനസ്സിലാക്കി പോകുന്നവര്‍ക്കെ
പ്രവാസം ഒരു ദു:സ്വപ്നമായി മാറാതിരിക്കൂ ....

' കടങ്ങള്‍ വീടിയിട്ടൊന്നും നാട്ടില്‍ പോക്ക് നടക്കില്ല ..! ' 2\. ' പെണ്ണ് കെട്ടിയാല്‍ പ്രവാസത്തില്‍ നിന്നൊരു മോചനം പിന്നെ ഉറപ്പിക്കണ്ട ' 3\. ' വിസയുടെ കാഷ് ഒരിക്കലും നമ്മള്‍ക്ക് മൊതലായി തോന്നൂലാ ' 4\. ' കുടുംബക്കാര്‍ക്കൊന്നും ഒരിക്കലും നമ്മുടെ സാലറിയുടെ കണക്ക് പറഞ്ഞ് കൊടുക്കരുത് ' 5\. ' ആരോടായാലും ഗള്‍ഫിലെ ഉള്ള സ്ഥിതി തുറന്ന്! പറയുക ഗമക്ക് ഇല്ലാത്ത കാര്യങ്ങള്‍ പറഞ്ഞാല്‍ അനുഭവിക്കും ' 6\. ' കയ്യഴഞ്ഞ് കൊടുക്കുമ്പോഴും ഒരു നുള്ള് അവനവന് വേണ്ടി മാറ്റി വെക്കുക ' 7\. ' വീടാവാതെ നിര്‍ത്തി പോകുന്നത് മനപ്രയാസം കൂട്ടും ' 8\. ' കുട്ടികളെയും ഭാര്യയേയും എല്ലാം ഗള്‍ഫിലെ പൈസ ഉണ്ടാക്കുന്നത് എങ്ങനെയാണെന്ന് പറഞ്ഞ് കൊടുത്ത് മനസിലാക്കുക ' 9\. ' പരമാവധി വിസ കച്ചോടം ഒഴിവാക്കുക ' 10\. ' കാഷ് കൊണ്ട് മുന്നോട്ട് പോകുന്ന കൂട്ട് കെട്ടുകള്‍ ഗള്‍ഫില്‍ ഒഴിവാക്കുക ' ഇങ്ങനെ കുറേയുണ്ട് ... ബാക്കി വൈകാതെ പറയാം ...! ഇതൊക്കെ മനസ്സിലാക്കി പോകുന്നവര്‍ക്കെ പ്രവാസം ഒരു ദു:സ്വപ്നമായി മാറാതിരിക്കൂ ....
